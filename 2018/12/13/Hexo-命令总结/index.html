Hexo 命令总结

Posted on 2018-12-13 | In Tools

复合部署流程

hexo clean  (清除临时文件，可以不加)
hexo g -d  (生成+部署)

草稿

hexo new draft "xxx"  新建草稿
hexo publish "xxx"  发布草稿

常用命令

hexo new "xxx"  新建文章
hexo publish "xxx"  新建文章
hexo generate  生成静态页面
hexo server  启动服务(预览)
hexo deploy  部署
hexo help  帮助
hexo version  查看版本

复合命令 & 简写

hexo s -g  生成+预览
hexo d -g  生成+部署

命令的简写为：

  • hexo n  ==  hexo new
  • hexo g  ==  hexo generate
  • hexo s  ==  hexo server
  • hexo d  ==  hexo deploy

本地测试

可以直接使用 hexo s -g
如果部分静态文件本地有效，部署后无效，本地启动服务后再重新部署即可（不需清除缓存
